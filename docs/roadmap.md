# Roadmap

## 0.1.x

- Improve local check coverage for Python, JavaScript, and Rust projects.
- Add configuration for project-specific check weights.
- Add SARIF or GitHub Step Summary output for CI.

## 0.2.x

- Scan public GitHub repository URLs using repository metadata.
- Add a GitHub Action wrapper.
- Compare readiness between two commits in a pull request.

## 0.3.x

- Add optional OpenAI-assisted issue triage.
- Generate pull request review checklists from changed files.
- Draft maintainer-support application evidence from real repository metadata.

## Principles

- No fabricated adoption metrics.
- Offline-first behavior remains available.
- LLM features are optional and clearly marked.
