# Draft Maintainer-Support Application Evidence

## Goal

Help maintainers prepare honest application evidence for maintainer-support programs.

## Proposed Behavior

Generate a local Markdown draft from real repository metadata, readiness checks, CI status, release history, and maintainer workflow files.

## Acceptance Criteria

- Does not fabricate stars, forks, downloads, or adoption claims.
- Clearly marks missing evidence.
- Produces a Markdown draft that maintainers can review before submitting.
- Documents responsible use and application limits.
