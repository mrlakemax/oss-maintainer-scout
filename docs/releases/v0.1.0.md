# v0.1.0 Release Notes

Initial public release of `oss-maintainer-scout`.

## Highlights

- Local repository scanner for open-source maintainer readiness.
- Weighted score and grade for common project health signals.
- Markdown and JSON report output.
- Checks for README, license, contributing guide, code of conduct, security policy, issue templates, pull request template, CI, tests, and changelog.
- Python CLI entry points:
  - `oss-scout scan .`
  - `python -m oss_maintainer_scout scan .`

## Notes

This release is intentionally offline-first and does not require API keys. Future releases may add optional GitHub URL scanning, GitHub Action output, and OpenAI-assisted maintainer workflows.
